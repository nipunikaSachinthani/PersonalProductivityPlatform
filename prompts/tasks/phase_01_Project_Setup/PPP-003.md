## 2. Objective

> Scaffold the React + Vite + TypeScript + Tailwind CSS + shadcn/ui frontend application.

This task creates the entire frontend skeleton — the React SPA that users will interact with for the remaining 11 phases. Vite provides fast dev server and build tooling; TypeScript enforces type safety; Tailwind CSS enables utility-first styling; and shadcn/ui provides accessible, composable UI primitives. This task teaches modern frontend tooling setup, design system integration, and the folder structure that will support feature-based organization.

## 4. Technical Guidance

**Conventions for this module:**
- Project scaffolded with `npm create vite@latest frontend -- --template react-ts`
- Tailwind CSS v3 configured via `tailwind.config.js` with content paths
- shadcn/ui initialized with `npx shadcn-ui@latest init` (default style, CSS variables)
- Folder structure matches ARCHITECTURE_ANALYSIS.md: `pages/`, `components/`, `features/`, `services/`, `hooks/`, `layouts/`, `routes/`, `types/`, `lib/`
- Install core dependencies: `react-router-dom`, `@tanstack/react-query`, `axios`, `react-hook-form`, `zod`, `@hookform/resolvers`
- `tsconfig.json` with strict mode enabled

**Concepts practiced in this task:**
- Vite build tooling
- Utility-first CSS with Tailwind
- Component library integration (shadcn/ui)
- TypeScript strict mode configuration
