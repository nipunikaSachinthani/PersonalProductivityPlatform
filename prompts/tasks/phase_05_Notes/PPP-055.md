## 2. Objective

> Integrate TipTap rich text editor with shadcn/ui toolbar for creating and editing notes.

The TipTap editor provides a rich-text editing experience — bold, italic, headings, lists, links — all rendered via a shadcn-styled toolbar. The editor outputs HTML (or JSON) that is sent to the backend and sanitized server-side. The note form (create/edit) embeds the TipTap editor below the title input, with category selection via shadcn `Badge` multi-select and optional project association via a dropdown. This task teaches rich-text editor integration, content serialization, and the create-edit form pattern for complex content types.

## 4. Technical Guidance

**Conventions for this module:**
- Editor: `@tiptap/react` with extensions: StarterKit (bold, italic, heading, bulletList, orderedList, blockquote, code, horizontalRule), Link, Placeholder
- Toolbar: shadcn `Toggle` buttons for each formatting option, styled with Tailwind; positioned above the editor
- Component: `RichTextEditor` in `features/notes/RichTextEditor.tsx` — controlled component accepting `content` and `onChange`
- Form: `NoteForm` with title input + TipTap editor + category multi-select (shadcn `Command` or `Badge` list) + project dropdown
- Content serialization: editor outputs HTML via `editor.getHTML()`; stored and sent to API
- Create/edit share the same `NoteForm` component (edit pre-populates from existing note)

**Concepts practiced in this task:**
- Rich-text editor integration (TipTap)
- Editor state management
- HTML content serialization
- Complex form composition (text + rich content + multi-select)
