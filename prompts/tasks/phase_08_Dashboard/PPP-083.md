## 2. Objective

> Integrate Recharts: bar chart for tasks by status, line chart for completions over time, with date range selector.

Recharts provides React-native charting components. The bar chart visualizes the status breakdown (Todo/InProgress/Done) with color-coded bars. The line chart plots daily completions from the chart data endpoint as a smooth trend line with dots at data points. A date range selector (7d, 30d, 90d buttons) above the line chart changes the range. Charts are responsive, have tooltips on hover, and match the app's color scheme. This task teaches charting library integration, responsive chart configuration, and interactive time-range filtering.

## 4. Technical Guidance

**Conventions for this module:**
- Library: `recharts` (`BarChart`, `Bar`, `XAxis`, `YAxis`, `Tooltip`, `ResponsiveContainer`, `LineChart`, `Line`, `CartesianGrid`)
- Components: `StatusBarChart`, `CompletionLineChart` in `features/dashboard/`
- Bar chart: XAxis = status labels, YAxis = count; bars colored by status (green=Done, blue=InProgress, gray=Todo)
- Line chart: XAxis = date, YAxis = completions; `type="monotone"` for smooth curve; `stroke="#10b981"` (green)
- Date range selector: shadcn `ToggleGroup` or `Button` group with "7d", "30d", "90d" options; selection drives chart data query param
- Both charts wrapped in `ResponsiveContainer width="100%" height={300}`
- Tooltip: custom tooltip showing formatted date and count
- Loading: skeleton chart placeholder; error: error message in chart area

**Concepts practiced in this task:**
- Charting library integration (Recharts)
- Responsive chart configuration
- Interactive date-range filtering
- Custom tooltip styling
