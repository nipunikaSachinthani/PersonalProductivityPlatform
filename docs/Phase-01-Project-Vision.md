# Personal Productivity Platform
## Project Vision Document

### Purpose
This project is designed as a complete software engineering refresher for an experienced .NET and React developer returning to professional development after a career break.

The goal is not merely to build a task management application, but to practice the complete software development lifecycle used in modern software teams.

---

# Business Problem

Users need a lightweight productivity platform that allows them to:

- Manage tasks
- Organize projects
- Store notes
- Track progress
- Receive reminders
- View productivity analytics
